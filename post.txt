Perhatian:
Persamaan yang berikut tidak sepenuhnya benar:
i=√−1
Walaupun persamaan itu berasal daripada persamaan
i²=−1
persamaan pertama mempunyai sebuah fungsi yang tidak terdapat dalam persamaan kedua iaitu fungsi punca kuasa dua. Definisi fungsi ini jelas dalam set nombor nyata tetapi tidak jelas dalam set nombor kompleks. Sebabnya fungsi punca kuasa dua ialah fungsi tidak selanjar dalam set nombor kompleks. Kesannya kalis berikut, walaupun sah dalam set nombor nyata, tidak sah dalam set nombor kompleks:
√(ab)=(√a)(√b)
Jika kalis ini digunakan dalam set nombor kompleks, kita boleh menulis bukti tidak sah yang berikut:
−1=(√−1)(√−1)=√(−1×−1)=√1=1
Fungsi pengeksponenan tiada masalah ini kerana definisinya jelas dalam set nombor kompleks. Oleh itu, persamaan kedua lebih tepat daripada persamaan pertama dan sah dalam semua keadaan.
Kesimpulannya, sila gunakan persamaan kedua demi ketepatan.

ڤرهاتين:
ڤرسامأن يڠ برايكوت تيدق سڤنوهڽ بنر:
i=√−1
والاوڤون ڤرسامأن ايت برأصل درڤد ڤرسامأن
i²=−1
ڤرسامأن ڤرتام ممڤوڽاءي سبواه فوڠسي يڠ تيدق ترداڤت دالم ڤرسامأن كدوا ايايت فوڠسي ڤونچا كواس دوا. دفينيسي فوڠسي اين جلس دالم سيت نومبور ڽات تتاڤي تيدق جلس دالم سيت نومبور كومڤليك س. سببڽ فوڠسي ڤونچا كواس دوا اياله فوڠسي تيدق سلانجر دالم سيت نومبور كومڤليك س. كسنڽ كاليس برايكوت، والاوڤون صح دالم سيت نومبور ڽات، تيدق صح دالم سيت نومبور كومڤليك س:
√(ab)=(√a)(√b)
جك كاليس اين دڬوناكن دالم سيت نومبور كومڤليك س، كيت بوليه منوليس بوقتي تيدق صح يڠ برايكوت:
−1=(√−1)(√−1)=√(−1×−1)=√1=1
فوڠسي ڤڠك سڤونينن تياد مسئله اين كران دفينيسيڽ جلس دالم سيت نومبور كومڤليك س. اوليه ايت، ڤرسامأن كدوا لبيه تڤت درڤد ڤرسامأن ڤرتام دان صح دالم سموا كادأن.
كسيمڤولنڽ، سيلا ڬوناكن ڤرسامأن كدوا دمي كتڤتن.
